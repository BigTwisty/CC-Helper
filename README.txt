CC-Helper
-----------

This tool is intended to make it easier/faster to copy working lua files to a computer in a local Minecraft session.

Features:

- Select World and Computer by name
- Easy access to the Computer directory
- Copies selected files from your project directoy to the computer
- Auto switching to the Minecraft window and closing of the tool after copy
- Auto running of a script after switching to Minecraft window
